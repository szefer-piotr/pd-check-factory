# React UI Release Checklist

- [ ] Install frontend dependencies in `frontend/`.
- [ ] Run `npm run lint` and resolve warnings/errors.
- [ ] Run `npm run test` and confirm primary path coverage passes.
- [ ] Run `npm run build` and verify successful bundle output.
- [ ] Validate mobile/tablet/desktop layout manually.
- [ ] Validate keyboard navigation and focus styling.
- [ ] Confirm loading, empty, error, and success states.
